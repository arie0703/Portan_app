# ポル単（仮）

ポルトガル語の単語帳アプリ(Flutterの練習も兼ねて・・)


## データベース
- words 単語を格納するテーブル
- wordfolders 単語帳を格納するテーブル
- word_belongings どの単語がどの単語帳に属しているかを管理するテーブル word_idとfolder_idを格納している
